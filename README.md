# one-percent-better-poker-site

Front door for `onepercentbetter.poker`.

This repo is the poker-specific surface inside the wider 1% Better system.
It should feel related to `onepercentbetter.dev`, but it is not a copy.
The public story here is narrower:

- poker analytics
- exploit review
- action and staking workflow tools
- serious-player product positioning

## Working Rules

- Keep the visual system related to `1% Better.dev`
- Use a darker, sharper, more instrument-like poker tone
- Prefer strong positioning over too many pages
- Do not overbuild before the poker tools themselves earn it

## Local Development

```bash
cd frontend
npm install
npm run dev
```

## Build

```bash
cd frontend
npm run build
```
