# one-percent-better-poker-site

Front door for `onepercentbetter.poker`.

This repo is the poker monetization surface inside the wider `1% Better` system.
It should feel related to `onepercentbetter.dev`, but it is not a copy.
The job here is simple:

- explain one poker product clearly
- make the offer easy to understand
- point to one primary CTA
- help validate whether the market will pay

## Working Rules

- Keep the visual system related to `1% Better.dev`.
- Use a darker, sharper, more instrument-like poker tone.
- Prefer one clear product story over a product family sprawl.
- Keep the page focused on monetization and product clarity.
- Do not overbuild before the poker tool itself earns it.

## Current Positioning

Primary product: `Exploit Better`

Core promise:

- one tournament
- three repeated habits
- one next-session adjustment

Secondary poker ideas can exist in the repo ecosystem, but this site should not dilute the main offer to explain them.
